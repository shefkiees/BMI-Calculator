# BMI Calculator

A simple and user-friendly BMI Calculator app built with React Native and Expo.  
The app supports both **Metric (kg/cm)** and **Imperial (lb/in)** units, calculates BMI,  
displays category (Underweight, Normal, Overweight, Obese) with colors,  
and keeps a **history log** of previous BMI calculations.

## Features

- Supports **Metric** and **Imperial** units
- Calculates BMI and displays category
- Color-coded BMI categories for easy visualization
- History log of previous BMI calculations

## How to Run

```bash
# Clone the repository
git clone https://github.com/shefkiees/BMICalculator.git

# Go to project folder
cd BMICalculator

# Install dependencies
npm install

# Start the app
npx expo start
```
